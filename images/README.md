# 재료 이미지 넣는 법

이 폴더에 아래 **4개 파일명 그대로** 이미지를 넣으면 자동 적용됩니다.
(파일이 없으면 기존 색 점으로 표시 — 앱은 그대로 동작)

| 재료 | 파일명 | 권장 |
|------|--------|------|
| 철 가루 | `iron.png` | 정사각형, 256×256 이상, 배경 단순 |
| 숯가루 | `charcoal.png` | 〃 |
| 소금 | `salt.png` | 〃 |
| 질석 | `vermiculite.png` | 〃 |

## 규칙
- **파일명/확장자 정확히** (소문자 `.png`). jpg 쓰려면 코드의 경로도 `.png`→`.jpg`로 바꿔야 함.
- 정사각형 권장 (버튼·주머니 입자가 정사각으로 잘림).
- 용량 작게 (각 200KB 이하 권장) — 크롬북 로딩 빠르게.
- 넣은 뒤 `index.html`/버전 파일은 수정 불필요. 그냥 커밋·푸시하면 GitHub Pages 반영.

## 적용 위치
1. 재료 버튼의 동그란 색 점 → 이미지
2. 주머니 안에 흩뿌려지는 입자 → 같은 이미지 미니 타일

## 추가로 더 넣고 싶으면
`index.html`의 `DEVICES.warmer.ingredients` / `DEVICES.cooler.ingredients` 각 항목에
`img:"images/파일명.png"` 한 줄 추가하면 됨. 예: 흡열의 질산 암모늄.
